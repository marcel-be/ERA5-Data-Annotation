# ERA5-Data-Annotation
A tool for integrating ERA5 climate reanalysis data with any spatio-temporal observations. Downloads ERA5 variables and precisely links them to your point-coordinate data (lat, lon, time) to bridge your measurements with detailed historical atmospheric conditions (point coordinates).

Download ERA5 data from Climate Data Store (CDS) of Copernicus Project
Use CDS API to fetch files


helpful code for Download and Annotation: GitHub: https://github.com/kamransafi/ERA5_move2
